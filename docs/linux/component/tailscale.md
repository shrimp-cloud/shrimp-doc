# Tailscale

> Tailscale 是一款基于 WireGuard 协议的零配置虚拟私有网络（VPN）工具，它能让你在全球各地的设备之间建立一个加密的私有局域网。无论这些设备是在家庭宽带、公司网络还是移动数据下，只要安装了 Tailscale，它们就能像在同一间屋子里一样互相访问。


## 基础信息

- 官网: https://tailscale.com/
- 登录: 使用 OAuth2 方式授权登录 (如 github)


# 注册使用

- Windows/Mac: 安装完软件后，客户端发起登录
- Linux: Console 控制台, 生成  Linux 安装命令，执行命令即可
- iOS/Android: 设备上安装客户端，登录即可


# 使用

- 完成注册的机器可以互访
