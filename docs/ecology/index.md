## 前端模块

> 前端模块主要完成具体的业务功能，分为以下几个端：
- console 总控制台，面向于开发/运维人员
- manager 总管理后台，面向于平台管理人员
- admin 租户/模块管理后台，面向于使用系统客户，租户
- customoer 用户终端，面向于终端用户
- miniapp 小程序终端，面向终端用户
