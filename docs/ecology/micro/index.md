# 微模块

> 微模块，指在建设包含特定弱业务的功能模块，在任意应用中，若需要使用相关功能模块，可快速的通过依赖 starter ，再完成数据库表创建/配置，应用配置，即可完成功能模块的快速集成。


## 微模块定位

- 微模块, 微小的，不可再拆分的功能单位。
- 微模块一般不涉及业务功能，但辅助于业务功能
- 微模块的底层，是框架，上层是业务模块
- 微模块不具备独立部署能力
